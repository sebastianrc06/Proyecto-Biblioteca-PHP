Título: Sistema de Gestión de Biblioteca.

Descripción: Aplicación web desarrollada en PHP nativo para el control de usuarios y préstamos de libros.

Funcionalidades:

Registro de nuevos usuarios.

Inicio de sesión con validación de credenciales.

Catálogo de libros interactivo.

Sistema de préstamos con envío de confirmación vía PHPMailer.
